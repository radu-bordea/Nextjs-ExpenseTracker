# Start the project
- npx create-next-app@latest expense-tracker --app --typescript --tailwind

# dependencies & libraries
-npm install @clerk/nextjs @hookform/resolvers react-hook-form zod zustand drizzle-orm pg lucide-react recharts
- npx shadcn@latest init
# Package	Purpose
@clerk/nextjs	🔐 Auth & user management
react-hook-form + zod → Form handling + validation
@hookform/resolvers	Connects zod with react-hook-form
zustand → Global state for expenses
drizzle-orm, pg → PostgreSQL ORM
shadcn/ui + lucide-react → Modern UI components
